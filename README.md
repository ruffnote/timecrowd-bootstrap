# TimeCrowd Bootstrap

Bootstrap4 Theme for TimeCrowd with Style Guidelines

## Install & Quick start

```
npm install
bundle install
npx gulp
```

## Style Guidelines

```
npx gulp webserver
open http://localhost:7070
```
